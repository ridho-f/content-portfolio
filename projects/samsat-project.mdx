Aplikasi Manajemen SKPD Digital berbasis Laravel ini dirancang khusus untuk meningkatkan efisiensi kerja pegawai SAMSAT dalam mengelola administrasi pajak daerah secara terpadu. Melalui integrasi langsung ke API Bapenda, pegawai cukup memasukkan nomor plat kendaraan untuk menyajikan data pemilik dan total tagihan secara instan, serta didukung fitur pencatatan pemakaian, pelaporan dokumen rusak, hingga ekspor laporan ke format Excel untuk kebutuhan laporan.
