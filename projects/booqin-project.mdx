Booqin adalah aplikasi booking online untuk studio foto yang saya kembangkan dengan Laravel (backend) dan React (frontend), dilengkapi integrasi WhatsApp API untuk notifikasi otomatis serta payment gateway untuk pembayaran online yang aman. Pelanggan dapat melakukan reservasi real-time, membayar langsung setelah booking, dan menerima reminder sebelum sesi dimulai. Admin memiliki dashboard untuk mengelola layanan, jadwal, transaksi, dan data pelanggan. Sistem ini dirancang untuk mencegah double booking dan mengotomatisasi operasional agar lebih efisien dan profesional.
