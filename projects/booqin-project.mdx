Booqin adalah aplikasi booking online untuk studio foto yang saya kembangkan menggunakan Laravel sebagai backend dengan integrasi WhatsApp API untuk notifikasi otomatis dan sistem payment gateway untuk pembayaran online yang aman dan terverifikasi. Aplikasi ini memungkinkan pelanggan melakukan reservasi jadwal secara real-time, melakukan pembayaran langsung setelah booking, serta menerima reminder otomatis sebelum sesi dimulai. Di sisi admin, tersedia dashboard untuk mengelola layanan, jadwal, transaksi, dan data pelanggan. Booqin dirancang untuk mencegah double booking, mengotomatisasi proses operasional, serta memberikan pengalaman reservasi yang modern, efisien, dan profesional.
